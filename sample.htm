Welcome

Welcome to the JSnip Sample page.

JSnip is a lightweight JavaScript based Snipping package, aimed at those who want to add interactive elements to their webpages but lack the necessary JavaScript knowledge to do so.

This version of Jsnip contains 5 snippet types.

  • JsnipTabs
  • JSnipShowHide
  • JsnipImageSwitcher
  • JsnipScrollToTop and JsnipScrollToAnchor

For basic usage instruction's please check out my article here.

For the JavaScript code behind it all, checkout my Github.

 

If you'd like to request a new snippet for the next version feel free to get in touch.

JSnip is an light weight javascript snippeting package.

JSnip can be used in valid HTML5 documents and has been tested to work in:

  • Chrome (full)
  • Firefox 3.5, 3.6, 4 and 5 (full)
  • IE9, IE8, IE7 (most*)
  • Android/IOS

* Limited CSS3 supported. Ajax functions only tested in IE9

The look and feel of JSnip snippets can be altered easily using CSS.

All Jsnip snippets are nestable, so you can place tabs in showHides, then another set of showHides, and tabs inside that if you feel the need.

How does it work?

The JSnip snippets are powered by a simple javascript library called base.js which i wrote along side the snippets themselves.

Along with a set of shorthand functions, helper methods (such as prepend,hasclass, addclass etc) JSnip also offers basic set of animtions, fade's, scrolls and slides as well as a nice little AJAX request wrapper.

A full commented version of base.js should be packaged along side the minified version included in this document

This is just here to show that ShowHides are happy to work inside tabs.
